Footer

Footer;

Descrição

Este método é usado para criar o rodapé da página. Ele é automaticamente chamado por AddPage() e Close() e não deve ser chamado diretamente pela aplicação. A implementação em FPDF está vazia, então você deve criar uma subclasse e sobrepor o método se você quiser um processamento específico.

Exemplo

TFPDFExt = class(TFPDF)
  protected
    procedure Footer; override;
end;

procedure TFPDFExt.Footer(APDF: TFPDFExt);
begin
  // Vai para 1.5 cm da parte inferior
  APDF.SetY(-15);
  // Seleciona a fonte Arial itálico 8
  APDF.SetFont('Arial','I',8);
  // Define cor do texto cinza
  APDF.SetTextColor(128);
  // Imprime o número da página corrente e o total de páginas
  APDF.Cell( 0, 10, 'Página '+IntToStr(APDF.PageNo())+'{nb}', '0', 0, 'C');
end;

PDF := TFPDFExt.Create;
PDF.SetAliasNbPages();
