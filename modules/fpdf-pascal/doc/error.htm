Error

Error(const ATextMsg: String; E: Exception = nil);

Descrição

Este método é chamado automaticamente caso ocorra um erro fatal; ele simplesmente dispara uma exceção com a mensagem informada.
Uma classe herdada pode substituí-lo para personalizar a manipulação de erro, mas o método não deve retornar null, caso contrário, o documento resultante provavelmente seria inválido.
Pode ser utilizada também para reproduzir a mensagem dentro de um tratamento de exceção Try/Except.

Parâmetros

ATextMsg
A mensagem de erro.
E
Exceção capturada quando utilizado dentro de um Try/Except. Caso utilizado este parâmetro a mensagem da exceção é concatenada com ATextMsg.
