SetAliasNbPages

SetAliasNbPages(const AAlias: String);

Descrição

Define um apelido para o número total de páginas. Ele será substituído quando o documento for fechado.

Parâmetros

AAlias
O apelido. Valor padrão: {nb}.

Exemplo


TFPDFExt = class(TFPDF)
  protected
    procedure Footer; override;
end;

procedure TFPDFExt.Footer(APDF: TFPDFExt);
begin
  // Vai para 1.5 cm da parte inferior
  APDF.SetY(-15);
  // Seleciona a fonte Arial itálico 8
  APDF.SetFont('Arial','I',8);
  // Define cor do texto cinza
  APDF.SetTextColor(128);
  // Imprime o número da página corrente e o total de páginas
  APDF.Cell( 0, 10, 'Página '+IntToStr(APDF.PageNo())+'{nb}', '0', 0, 'C');
end;

PDF := TFPDFExt.Create;
PDF.SetAliasNbPages();
